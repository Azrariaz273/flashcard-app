# flashcard-app
A responsive Flashcard App built with HTML, CSS, and JavaScript featuring add, edit, delete, navigation, and local storage.
# ⚡ Flashcard Quiz App
A clean and responsive web application designed for active recall and efficient studying.

## 🚀 Features
- **Create & Manage:** Add, Edit, and Delete flashcards easily.
- **Interactive:** Flip cards to reveal answers with smooth CSS transitions.
- **Persistent Storage:** Data is saved in the browser's LocalStorage.
- **Responsive:** Optimized for mobile and desktop screens.

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox/Grid)
- Vanilla JavaScript

## 💡 How to use
1. Add a Question and Answer using the form.
2. Click the card to flip it and view the answer.
3. Use the Next/Prev buttons to navigate.
4. Your data persists even after refreshing the page!
